## 二、产品定位

- **核心理念**：理性消费复盘 + 社区评价分享
- **目标用户**：18-28岁年轻白领/学生党
- **商业模式**：社区内容变现 + KOL带货
- **设计风格**：新粗野主义（Neo-Brutalism）— 粗犷边框、绿黑撞色、高对比度、Z世代叛逆感

## 三、设计风格定义

```
主色：绿色系（具体色值待定）
撞色：绿 × 黑为主，辅以大胆高饱和点缀
边框：粗实线（3-5px），硬阴影（offset shadow，无模糊）
字体：大胆、粗体标题，清晰正文
圆角：极小或无圆角（brutalism特征）
卡片：实色背景 + 粗边框 + 硬投影
整体：简约现代，信息层次清晰，视觉冲击力强
```

## 五、技术栈（计划）

```
框架：      React 18 + Next.js（或 Vite + React）
样式：      Tailwind CSS
动效：      framer-motion
图表：      recharts
语音：      Web Speech API（浏览器原生）
存储：      LocalStorage（全站数据基础）
表单：      FormSubmit（免费，无需后端）
部署：      Vercel（免费，满足任务书"上服务器"要求）
```

## 六、五大核心交互

| 动效 | 位置 | 实现方式 |
|------|------|----------|
| 卡片堆叠滚动 | 首页 | framer-motion + scroll event |
| 横向无限滑动 | 首页"避坑宣言" | framer-motion infinite marquee |
| 手风琴折叠 | 服务介绍/玩法说明 | framer-motion AnimatePresence |
| 数字增长动画 | 统计分析页 | framer-motion + useInView |
| 表情评分系统 | 写日记页/列表页 | emoji按钮组 + state管理 |

## 七、核心功能模块

1. **日记CRUD**：商品信息记录、消费动机、使用体验、评分 → LocalStorage
2. **数据统计**：消费总额、理智指数、品类分布、存钱罐 → 基于LocalStorage计算
3. **语音交互**：搜索/评论/输入全场景，绿色麦克风icon贯穿全站（可选，Chrome优先）
4. **游戏化**：挑战任务、省钱排行榜、成就徽章
5. **响应式**：电脑/平板/手机三端适配
6. **表单提交**：FormSubmit，提交后原地显示成功提示（不跳转）

## 十、Figma方案

任务书要求交付Figma源文件。计划方案：
- 先完成前端代码实现
- 生成SVG素材（icon、装饰元素）可导入Figma
- 在Figma中手动搭建页面（保证Auto Layout、组件、图层命名规范）
- SVG直接导入Figma再导出.fig的方式**可用但有局限**（无Auto Layout、无组件系统、图层命名混乱），不建议作为主要方案
